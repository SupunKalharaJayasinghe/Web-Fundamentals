# Module 1: Web Foundations & Development Environment

## Learning Objectives:

- Understand how browsers, servers, and the internet work together
- Set up and use professional development tools
- Master browser DevTools for debugging
- Learn version control with Git and GitHub

## Core Topics:

- HTTP basics, URLs, and how the web works
- Code editors (VS Code) and browser DevTools
- Terminal/command line fundamentals
- Git workflow: commits, branches, merges, pull requests
